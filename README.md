📊 Bike Sales Customer Insights Dashboard (Excel)

📌 Project Overview

This project presents an interactive Excel dashboard analyzing customer demographics and behavioral patterns influencing bike purchases.

The dashboard enables dynamic filtering using slicers and provides insights into income, age, commute distance, and regional trends.

📁 Dataset Description

The dataset includes:

Customer demographics (Age, Gender, Marital Status)

Socio-economic factors (Income, Education, Occupation)

Lifestyle indicators (Cars, Commute Distance, Home Ownership)

Target variable: Purchased Bike (Yes/No)

🎯 Business Objective

Identify factors influencing bike purchases

Understand customer segmentation

Provide insights for targeted marketing strategies


📊 Dashboard Features

Average Income Analysis

Compares income levels of customers who purchased vs did not purchase bikes

Age Group Analysis

Shows purchase trends across different age brackets

Commute Distance Analysis

Examines how travel distance impacts buying decisions

Interactive Filters (Slicers)

Marital Status

Region

Education

🛠 Tools & Techniques Used

Microsoft Excel

Pivot Tables

Pivot Charts

Slicers

Data Cleaning & Transformation


🔍 Key Insights
Middle-aged customers show the highest purchase rate

Customers with higher income are more likely to purchase bikes

Short to moderate commute distances influence purchase decisions

Regional variations impact sales trends


💡 Conclusion

This dashboard helps businesses understand customer behavior and can be used to optimize marketing strategies and improve sales targeting.
